Bookshelf

To set up Bookshelf, just add a media library in the server configuration and set the content type to books. If you want to see support for a specific provider we always welcome code contributions!

Supported Formats

Please take in mind that his is not a complete list.
  • epub
  • mobi
  • pdf
  • cbz
  • cbr
  • mp3
  • m4a
  • m4b
  • flac

Offline Metadata providers

  • Open Packaging Format (OPF)
  • Calibre OPF
  • ComicInfo
  • ComicBookInfo
The following limitations apply:
  • .cbr Comics tagged with ComicRack's ComicInfo format are partially supported. Any metadata bundled with the comic book itself will be ignored. External files will be read.
  • The Advanced Comic Book Format format is not supported.
  • The CoMet format is not supported.

Online Metadata providers

  • Google Books
