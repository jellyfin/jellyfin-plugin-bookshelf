Bookshelf

To set up Bookshelf, just add a media library in the server configuration and set the content type to books. This plugin supports standard OPF as well as Calibre OPF if you'd like to import the metadata from another program. If you want to see support for a specific provider we always welcome code contributions!

Supported Formats

  • epub
  • mobi
  • pdf
  • cbz
  • cbr
The following limitations apply:
  • .cbr Comics tagged with ComicRack's ComicInfo format are partially supported. Any metadata bundled with the comic book itself will be ignored.
  • The Advanced Comic Book Format format is not supported.
  • The CoMet format is not supported.
